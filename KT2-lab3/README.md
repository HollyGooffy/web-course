# Криптовалютный Трекер (CryptoDash)

Простое веб-приложение для отображения данных о криптовалютах из CoinGecko API в табличном формате.

## Описание проекта

Проект представляет собой минималистичное приложение для мониторинга криптовалют. Данные получаются через API и отображаются в удобной таблице с возможностью сортировки, фильтрации и пагинации.

### Основные возможности

- **Табличное отображение**: Данные о криптовалютах в таблице с 6 столбцами
- **Работа с API**: Интеграция с CoinGecko API
- **Поиск**: Фильтрация монет по названию или символу
- **Сортировка**: По капитализации, имени, цене, объему или изменению за 24ч
- **Пагинация**: Настраиваемое количество элементов на странице (20/50/100)
- **Детальная информация**: Просмотр подробных данных о каждой монете
- **Адаптивный дизайн**: Корректное отображение на всех устройствах

### Столбцы таблицы

1. **#** - Ранг по рыночной капитализации
2. **Монета** - Название, символ и иконка криптовалюты
3. **Цена** - Текущая цена в USD
4. **24ч %** - Процентное изменение цены за 24 часа
5. **Капитализация** - Общая рыночная капитализация
6. **Объем (24ч)** - Объем торговли за последние 24 часа

## Технологии

### Frontend
- **React 19** - библиотека для UI
- **TypeScript** - типизация
- **Vite** - сборщик
- **React Router** - маршрутизация
- **Context API** - управление состоянием
- **Tailwind CSS** - стилизация
- **Radix UI** - UI компоненты
- **Axios** - HTTP-клиент
- **Recharts** - графики

### Backend
- **Express.js** - прокси-сервер
- **Node Cache** - кэширование запросов

### API
- **CoinGecko API** - источник данных

## Установка и запуск

### Требования
- Node.js 18+
- pnpm (или npm/yarn)

### Установка

```bash
pnpm install
```

### Запуск

```bash
pnpm dev
```

Приложение будет доступно:
- Frontend: http://localhost:5173
- Backend: http://localhost:3001

### Сборка

```bash
pnpm build
```

## Особенности

- **Минималистичный дизайн**: Только необходимый функционал
- **Кэширование**: Backend кэширует запросы на 5 минут
- **Производительность**: Оптимизированная загрузка данных
- **Простота**: Нет сложных зависимостей и избыточного кода
