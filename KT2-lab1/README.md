# МедЦентр Здоровье - PHP версия

Лабораторная работа №3 (КТ2) - Сайт медицинского центра с использованием PHP и базы данных.

## Реализованный функционал

✅ **Информация из базы данных:**
- Новости на главной странице загружаются из таблицы `news`
- Специалисты на главной странице загружаются из таблицы `doctors`

✅ **Форма записи к врачу:**
- Страница `appointment.php` с формой записи
- Данные сохраняются в таблицу `appointments`
- Валидация полей формы
- Сообщения об успехе/ошибке

✅ **Шаблоны (templates):**
- `includes/header.php` - шапка сайта
- `includes/footer.php` - футер сайта
- Используются на всех страницах

✅ **Текущая дата и день недели:**
- На главной странице выводится текущая дата
- День недели на русском языке (используется PHP)

## Установка и настройка

### 1. Требования
- PHP 7.4 или выше
- MySQL 5.7 или выше
- Веб-сервер (Apache/Nginx) или локальный сервер (OpenServer, XAMPP, Denwer)

### 2. Установка базы данных

1. Откройте phpMyAdmin или другой клиент MySQL
2. Импортируйте файл `database.sql`:
   ```sql
   mysql -u root -p < database.sql
   ```
   Или скопируйте содержимое файла и выполните в phpMyAdmin

3. База данных `medcentr_db` будет создана автоматически с тестовыми данными

### 3. Настройка подключения к БД

Откройте файл `config.php` и при необходимости измените параметры подключения:

```php
define('DB_HOST', 'localhost');     // Хост БД
define('DB_NAME', 'medcentr_db');   // Имя БД
define('DB_USER', 'root');          // Пользователь
define('DB_PASS', '');              // Пароль
```

### 4. Запуск проекта

#### Вариант 1: Встроенный PHP-сервер
```bash
cd KT2-lab1
php -S localhost:8000
```
Откройте в браузере: http://localhost:8000

#### Вариант 2: OpenServer/XAMPP
1. Скопируйте папку `KT2-lab1` в директорию `domains` (OpenServer) или `htdocs` (XAMPP)
2. Откройте в браузере: http://localhost/KT2-lab1

## Структура базы данных

### Таблица `news` (Новости)
- `id` - ID новости
- `title` - Заголовок
- `image` - Путь к изображению
- `date` - Дата публикации
- `content` - Текст новости
- `created_at` - Дата создания записи

### Таблица `doctors` (Специалисты)
- `id` - ID врача
- `name` - ФИО
- `specialty` - Специальность
- `experience` - Стаж работы
- `photo` - Путь к фото
- `created_at` - Дата создания записи

### Таблица `appointments` (Записи к врачу)
- `id` - ID записи
- `patient_name` - ФИО пациента
- `phone` - Телефон
- `email` - Email
- `doctor_id` - ID врача (внешний ключ)
- `appointment_date` - Дата приема
- `appointment_time` - Время приема
- `message` - Комментарий
- `created_at` - Дата создания записи

## Использованные технологии

- **PHP 7.4+** - серверная логика
- **MySQL** - база данных
- **PDO** - безопасная работа с БД (prepared statements)
- **HTML5** - разметка
- **CSS3** - стилизация (градиенты, анимации, grid, flexbox)

1. **Безопасность:**

   - Экранирование вывода через `htmlspecialchars()`
   - Валидация данных формы

2. **Шаблонизация:**
   - Переиспользуемые компоненты (header, footer)
   - Динамическое формирование заголовков страниц

3. **PHP-функционал:**
   - Вывод текущей даты и дня недели на русском
   - Динамическая загрузка данных из БД
   - Обработка форм с валидацией
